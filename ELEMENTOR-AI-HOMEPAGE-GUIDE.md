# Premium AI Consultant Homepage

Use this as the new homepage structure in Elementor. The goal is fewer cards, stronger hierarchy, more whitespace, and a more premium consulting feel.

## Section Order

1. Hero
2. Trust strip
3. Services
4. Process
5. Why Digit Waves
6. Final CTA

## Global Notes

- Avoid more than one section with boxed cards.
- Use dark backgrounds only for hero and final CTA.
- Keep section spacing generous.
- Use short copy blocks and stronger headlines.
- Reuse one primary button style everywhere.

## CSS Classes To Apply In Elementor

Section classes:

- `dw-premium-section`
- `dw-premium-hero`
- `dw-premium-trust`
- `dw-premium-services`
- `dw-premium-process`
- `dw-premium-proof`
- `dw-premium-cta`

- `dw-proof-list`
